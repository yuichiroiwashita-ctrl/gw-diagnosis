# gw-diagnosis

## プロジェクト概要
商談時に導入効果測定を出す診断フォーム。GitHub Pages で公開し、商談時にヒアリングした内容を入力して診断結果を表示し、効果測定を説明するために使用する。

## デプロイ
- GitHub Pages で公開

## 開発ルール

### コード実装・編集後の必須フロー
1. 実装・編集内容のコードレビュー（ロジック・UI・バグの確認）
2. ブラウザ動作確認（診断フォームの入力〜結果表示の一連フローをテスト）
3. 問題なければ GitHub にプッシュ

### README
全機能が出来上がった時点で、仕様・使い方・フォーム項目・診断ロジックをまとめて README.md に記載する。実装途中は README を更新しない。
